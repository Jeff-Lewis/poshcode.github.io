# Sith

### [download](//scripts/2650.ps1)

The VBS script is intended to audit an Active Directory Domain Controller by the events that it generates in the event log. Most usefull events are harvested and piped out into a report that will also generate a summary report of the main report and also e-mail the report to an admin.

Directories that will be created when running this script are:

C:\audit_AD [Reports are generated in this folder temporarily - Temp folder]

The script can be used to run a a daily scheduled task and Additional scripts can be programmed to automatically move these daily reports to a secure archive (I also have secondary scripts to this and will post upon request)

IMPORTANT = The security event log will also be backed up and cleared, the security event log will also be placed in the mentioned folder in a date format.

This Script is SPECIFICALY for server 2008! The event ID's are different in server 2003. The version of this script for Server2003 will be published upon request.

Lines 797-800 must be set with your own mail attributes

NOTE: The script can take up to to anything from 5minutes to 30minutes, depending on the size and amount of security entries in the security event log of the DC. Thus the event log is cleared daily and also backed up and moved. By doing this daily, the script runs faster incrementally.
