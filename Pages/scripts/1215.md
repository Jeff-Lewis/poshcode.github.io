# Get-StringRange

### [download](//scripts/1215.ps1)

Works like the integer range operator "..", but for strings.  Technique by Joel Bennett.

```posh
function Get-StringRange ( $Start, $End ) {
	[char[]]( [int][char]$Start..[int][char]$End )
}
```
