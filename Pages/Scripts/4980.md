# Get-ClipboardText

### [download](Scripts\4980.ps1)

This function retrieves the (Unicode) text from the clipboard.

```posh
Function Get-ClipboardText
{
    [CmdletBinding()]
    [OutputType([String])]
    
    Param() # No parameters
    
    [System.Windows.Forms.Clipboard]::GetText( 'UnicodeText' ) | Write-Output
}
```
