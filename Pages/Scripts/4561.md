# directories tree

### [download](Scripts\4561.ps1)

If 'tree' pocket has not been installed that you can use next script but note that it gets only directories.

```posh
#!/bin/bash

if [ -z "$1" ]; then loc=$(pwd); else loc=$1; fi
ls -aR $loc | grep ':$' | sed -e 's/:$//;s/[^-][^\/]*\//--/g;s/^/ /;s/-/|/'
```
