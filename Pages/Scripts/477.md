# Ellipsis

### [download](Scripts\477.ps1)

The infamous ellipsis function lets you pick out a single property, rather like using Select -Expand ... except it runs in about 2/3 the time.

```posh
## Usage: ls | ... FullName
################################################
${function:...} = { process { $_.$($args[0]) } }
```
