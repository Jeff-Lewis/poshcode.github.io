# A tip to validate IPAddr

### [download](Scripts\4447.ps1)

Another tip, really short, to validate an IP address passed to a function.

```posh
param
(
	[Parameter(Mandatory=$True)]
	@@[ValidateScript({$_ -match [IPAddress]$_ })]
	[String]$ip
)
```
