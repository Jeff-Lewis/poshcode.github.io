# Get-HarddriveModel

### [download](Scripts\5285.ps1)

Retrieves harddrive model name without WMI

```posh
function Get-HarddriveModel {
  <#
    .NOTES
        Author: greg zakharov
  #>
  (gp (
      Join-Path $key (
        'Enum\' + (gp (Join-Path ($key = 'HKLM:\SYSTEM\CurrentControlSet') 'Services\Disk\Enum')).('0')
      )
    )
  ).FriendlyName
}
```
