# Convert-ToCHexString

### [download](1153.ps1)  [1154](1154.md)

Converts an input string into its C ASCI Hex equivalent, useful for converting a string into an unsigned char array

```posh
function Convert-ToCHexString {
  param([String] $input
  $ans = ''

}
```
