# Is-Natural

### [download](3593.ps1)"

This is a small little function I did to test and see if a number is a natural number or not. It returns true or false. 

```posh
function is-natural{
	param ($number)
	if($number -like "*.*" -or $number -like "*-*"){
		return $false
	}else{return $true}
}
```
