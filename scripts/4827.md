# Bash: file full path

### [download](4827.sh)  

Prints full path for each file in a directory

```bash
#!/bin/bash

for file in $(ls -a "$@"); do
	echo -n $(pwd)
	[[ $(pwd) != "/" ]] && echo -n /
	echo $file
done
```
