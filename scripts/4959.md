# local users SIDs

### [download](4959.ps1)



```posh
@(gci (Split-Path $env:userprofile) | ? {
  $_.Name -ne 'All Users'
}) | % {
  '{0, 13}: {1}'-f $_.Name, (New-Object Security.Principal.NTAccount(
    (Join-Path ([Environment]::MachineName) $_.Name)
  )).Translate([Security.Principal.SecurityIdentifier])
}
''
```
