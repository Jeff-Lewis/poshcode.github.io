# ip wlan adapter

### [download](5457.ps1) [parent](5456.md) 



```posh
$key = 'HKLM:\SYSTEM\CurrentControlSet'

(gp ($key + '\Services\Tcpip\Parameters\Interfaces\' + `
(Split-Path -Leaf (gp ($key + '\Control\Network\*\*\Connection') | ? {
  $_.MediaSubType -eq 2
}).PSParentPath))).DhcpIPAddress
```
