# 

### [download](5159.ps1)"

Copy files to dated directory.

```posh
$date = Get-Date -Format "yyyyMMdd"
$source = 'C:\dir'
$destination = "C:\someotherdir\$date\"

New-Item -ItemType directory -Path $destination
Copy-Item $source "$destination" -Recurse -Force
```
