# Get-HarddriveModel

### [download](5293.ps1) [parent](5285.md) 

Retrieves harddrive model name without WMI

```posh
function Get-HarddriveModel {
  <#
    .NOTES
        Author: greg zakharov
  #>
  $key = 'HKLM:\SYSTEM\CurrentControlSet'
  (gp (
      Join-Path $key (
        'Enum\' + (gp (Join-Path $key 'Services\Disk\Enum')).('0')
      )
    )
  ).FriendlyName
}
```
